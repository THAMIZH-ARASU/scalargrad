# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.3] - 2026-02-13

### Added
- Initial release of ScalarGrad
- Core automatic differentiation engine with Scalar class
- Reverse-mode backpropagation implementation
- Neural network components (Neuron, Layer, MLP)
- Optimizers: SGD with momentum and Adam
- Loss functions: MSE and SVM loss
- Training utilities with Trainer class
- Visualization tools for computation graphs and training metrics
- Comprehensive test suite
- Example scripts demonstrating functionality
- Full documentation and API reference

### Features
- Zero-dependency core functionality
- Support for basic arithmetic operations (+, -, *, /, **)
- Activation functions (ReLU, tanh, sigmoid, exp, log)
- Configurable logging and random seed
- Gradient clipping support
- Training history tracking
- Decision boundary visualization
